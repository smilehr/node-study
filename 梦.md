##### 一次梦里的故事

###### 男主
```
他是来乡村里支教的大学生，在支教的同时，并准备着考研的事情。幽默，风趣，却是个不折不扣的母胎solo。

谁也不知道他的真实家庭环境，但他好像也不想跟别人提起他的家庭，也不想太过依靠家里。所有的开支，都是由自己支教的那点绵薄工资负责；

经典的月光族，但他的薪水并非乱用，而是会买些自己感兴趣的书籍，或者是为学生添补些家用

对感情的木讷，对爱情的憧憬，导致他即使遇见心动的女孩子，也不会轻易冒进，而是后知后觉
```

###### 女主
```
她只是个生活并不如意，选择来乡村教书，逃避生活的，已毕业的女大学生。

温柔，平静似水，待人友好，但仿佛是尘世中飘然独立的一片叶。待人温和的里面，却是不为人知的，对于生活的迷茫

空闲时候的书籍、观看星空的望远镜、偶尔会跃出的古琴音符

而这一切之下却是对于爱情的执着
```

###### 女二
```
她只是个npc。学校里又一个不一样的人

严谨，却又受男主幽默感染，而被男主吸引，但只是纯净的朋友之谊

也有人问过她，世界上真的有纯粹的男女关系么  她回答有，但其实她自己也不知道

男女主之间的桥梁，甚至是助推手
```

###### 男二  （楠木可依）
```
女主的学弟，却受女主影响，一直跟着女主的脚步
```

###### 故事简介
```
随着他的自我介绍，他正式加入了这个学校，这个班级，当了一名语文老师。他的博学多才，幽默风趣，是他与学生打成一片的基础。办公室里，他轻松与老师们打成一片。

一次偶然的机会，他与严谨的主任，有了一次关系升级，从普通同事到朋友（可以是探讨工作时的共同点，亦或者是俩人相似的家庭环境和来乡村教书的相同想法）

有一天，休假的女主回到办公室，仿佛命运的牵引一般，男主被女主身上独特的性质而吸引。平时不爱表现的他，突然在工作中或者是在休息中，总爱在女主面前表现自己，但又不敢主动迈出第一步。

夜晚里，他发现了在楼顶抬头望天的她，俩人从初始的尴尬，到熟络，到畅聊，意犹未尽的俩人各回各自的宿舍，开始了这真正的第一步

慢慢的，俩人之间的交集越来越频繁，关系不断的升温。当男主觉得一切好像到那个点的时候，便策划了一次关系升级的事件，但意外的，遭到了女主的拒绝。

女主的不知所措，对爱情的敏感，逃离了那个他精心设计的场面

他以为她被他吓到，或者是并没有那种想法，所以，他尴尬了，性格里的怯懦，导致他和她回到了最初的关系（普通同事关系）

而她却并不是，也许告白场面让她惊到了，但后面越想越多后，剩下的只是惊喜。只是素来的一个人生活，加上遗世独立的性格，使她不好意思再去向男主说出她的真实想法

而这一切，都被女二看在眼里

关系破冰，确定关系，俩人感情迅速升温。但却从未越雷池一步

考试的时候快到了，这一次，俩人如之前一样，一起去都考试的城市，约好了考完一起好好的逛逛那个城市，然而一次意外，俩人都差点命丧黄泉

男主更严重的伤势，使得男主的家人不得不带他去更好的医院获得更好的治疗，而留给女主的，却是一句，他好了后自然会选择是否回来找你。

时光荏苒，又一次的天台上弹着古琴看宇宙，细微的动静，她转身看向顶楼的门处
```
